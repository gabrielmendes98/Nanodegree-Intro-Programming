World Wide Web:
É um sistema disponivel para ser acessado na internet, o qual é apresentado em forma de texto, imagens e sons. Ao acessarmos esse sistema utilizamos o protocolo HTTP, que informa ao servidor de um site que alguem está acessando-o, após isso ele reenvia os dados ao navegador do usuário em forma de HTML o qual é a linguagem que o computador entende e consegue formatar na tela do usuário.
